# DiskForge-Simulator

DiskForge is a web-based simulator to visualize disk scheduling algorithms like:

- FCFS
- SSTF
- SCAN
- C-SCAN

## Features
- User input for disk requests
- Graphical visualization
- Performance comparison
- Clean UI

## Tech Used
- HTML
- CSS
- JavaScript

## How to Run
Open index.html in browser
